# Clash for Android

* `系统要求：Android 4.4 及以上`
* `设备要求：Android Phone / Tablet`

{% hint style="info" %}
Clash for Android 不兼容 SSR 协议
{% endhint %}

1. 登入 Dler Cloud 用户中心
2. 点击「一键导入」按钮或手动复制托管地址导入
